# Talon Pilot Studio

**你的 AI 数字员工指挥台。**

把需求说给 Talon，它带着一支 AI 数字员工团队帮你把活干完 —— 从拆解任务、分工执行，
到产物预览、验收，全在一个界面里。

## 它能做什么

- **像聊天一样提需求** —— 你说要做什么，`@talon` 作为主持人自动拆解任务、安排合适的
  数字员工去做，全程帮你推进，你只管提需求和拍板。
- **一支随叫随到的 AI 团队** —— 创建你的数字员工（产品、前端、后端、测试、运维……），
  他们真实地帮你写代码、做东西、交付成果。
- **边做边看** —— 任务进行中就能在界面里实时预览做出来的网页、代码和文件，改完即看。
- **东西在你自己机器上** —— 客户端让你的电脑成为执行节点，代码和成果都在本地，
  你的数据归你。

## 怎么开始

1. 下载安装 Talon Pilot Studio
2. 打开客户端，登录你的账号
3. 在群聊里把需求告诉 Talon，剩下的交给它和你的数字员工团队
