# Changelog

发版说明的唯一真相源。发版时 `release.yml` 会抽取对应版本段落,**同时**用于 GitHub Release
正文和应用内更新弹窗(`latest.json` 的 `notes`)——一处维护、两处一致。

格式约定:每个版本一个 `## <版本号>` 段落(可带日期),正文用 Markdown(列表/加粗/链接均可,
应用内弹窗会渲染 Markdown)。版本号与 tag(去掉前导 `v`)一致。

## 0.1.6

- 修复 **tp-agent** 安装后检测不到的问题(`install`/`locate` 路径闭环,Windows/macOS 回退目录也覆盖)
- 帮装/登录/状态命令移出主线程,下载加超时与进度,断网不再卡死界面
- 新增 daemon 管理:启动 / 停止 / 重启,多账号切换后可一键重启生效
- 更新弹窗改为品牌化样式:Logo 居中、**Markdown 渲染**、滚动渐隐、下载进度条
- 自动更新与 tp-agent 下载支持镜像加速(可在配置里指定,大陆直连更稳)
- 开启内容安全策略(CSP),收紧脚本来源
- 新增系统托盘、外链走系统浏览器、单实例聚焦、窗口尺寸/位置记忆
